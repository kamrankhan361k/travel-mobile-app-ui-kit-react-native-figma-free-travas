B) ReactNative Setup Environments:

Download ReactNative file from link as your system type:
(Windows, Mac OS, Linux)

Link:


https://reactnative.dev/docs/environment-setup

After Download set file as your requirements file path, after setup done, use below command to check ReactNative Version setup is done or not.

Command:

react-native -v

C) Project Setup & Structure:

A. Windows:

1. Open an Administrator Command Prompt (right click Command Prompt and select "Run as Administrator"), then run the following command:

choco install -y nodejs-lts microsoft-openjdk11


2. Download Android Studio or Visual Studio Code.

3. Running your React Native application

Step 1: Start Metro

npx react-native start


Step 2: Start your application

npx react-native run-android



B. MacOS:

1. Node & Watchman : Run Command

a. brew install node
b. brew install watchman

2. Ruby

a. ruby --version
b. Install XCode Software

3. Running your React Native application

Step 1: Start Metro

npx react-native start


Step 2: Start your application

npx react-native run-ios
